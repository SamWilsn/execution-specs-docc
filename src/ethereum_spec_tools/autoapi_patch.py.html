    ethereum_spec_tools.autoapi_patch

    The autoapi function used for documentation does not fully capture assignment of object variables.

    E.g- in the module test.py A.x = 10 B.x = 5

    Both instances are treated as test.x instead of test.A.x & test.B.x This could incorrectly lead to duplicates.

    This tool is used as a patch.

    get_assign_value_patch
